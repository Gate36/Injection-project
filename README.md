# Injection-project
SSLC Penetration_Testing Project about SQL Injection

## 프로젝트 목표
    SQL Injection을 사용하여 지정된 목표 사이트 모의해킹
    파이썬을 이용한 자동화 프로그램 제작
    그 후, 전체 과정 및 정리된 보고서 제출

## 공격 목표
    SQL Injection JSP 문제 7번

## 필수 공격 요소
    1. 전체 테이블명
    2. ANSWER table의 컬럼명
    3. ANSWER table의 데이터

## 추가 요소
    1. 테이블 다 뽑아놓고 고르게 하기
    2. member 정보
    3. UI/UX 구성
    4. 엑셀 추출

## 보고서 조건
    - 보고서: 양식 자유, 단, 양식이 없는 새 파일에서 시작
    - 첫 번째 장: 표지
    - 두 번째 장: 목차 (목차, 그림목차, 표목차)
    - 그림목차, 표목차 필수 아님

## 보고서 흐름
    1. 개요/기획/서론
    2. 개발 환경/소프트웨어
    3. 기능 기획/소개
    4. 실행 결과
    5. 프로젝트 후기
    - (별첨. 소스코드 결과)
